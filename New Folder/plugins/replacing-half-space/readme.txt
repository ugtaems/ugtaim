== Description ==

Virastar developed a new version of the plugin is Replacing half space.
This plugin some problems to solve in Farsi.

Some Features:

* Replace Half Space in Content, Comment and Title.
* Replace English number to Farsi number.
* Replace Arabic number to Farsi number.
* Replace Arabic words to Farsi words.
* Show Virastar In Theme.
* Backlinks for Any Words (Unlimited).

Langusgaes:

* Persian
* English [Mostafa Soufi](http://iran98.org/)

Send email for Translation files: mst404[a]gmail[dot].com

== Installation ==

1. Upload `replacing-half-space.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To display the editor in your blog, put <code><?php virastar(); ?></code> in theme.
